# MEYOS — Корпоративный сайт ассоциации мебельщиков Узбекистана

Статический многостраничный сайт некоммерческой ассоциации MEYOS с переключением **4 концепций дизайна × 3 цветов × 3 языков**.

## Структура

```
├── index.html            Главная (все блоки + hero с динамикой)
├── about.html            О компании
├── residency.html        Резидентство и льготы
├── partners.html         Партнёры и резиденты
├── programs.html         Программы и EduJob
├── news.html             Новости
├── events.html           Мероприятия
├── contacts.html         Контакты
└── assets/
    ├── css/themes.css    Все стили + темы
    └── js/
        ├── i18n.js           Словарь RU/UZ/EN (основные ключи)
        ├── i18n-auto.js      Auto-перевод по text-node (~300 строк)
        ├── theme-switcher.js Переключатель концепций и цветов
        └── main.js           Форма-модалка, фильтры, news popup
```

## Концепции

| ID | Название | Формат | Совместимые цвета |
|----|----------|--------|-------------------|
| `flow` | Экосистема | Минимализм, воздушный | green, wood, sapphire |
| `corporate` | Корпоративный | Classic split | green, sapphire |
| `forge` | Кузница | Dark industrial, BENTO | sapphire, wood |
| `sovereign` | Суверен | Gov, Playfair, BENTO | wood, sapphire |

## Цвета

- `green` — мягкий лесной зелёный
- `wood` — благородное дерево / медь
- `sapphire` — глубокий синий / золото

Виджет переключения — плавающая панель внизу по центру.

## Функционал

- Переключатель цветов × концепций (9 комбинаций)
- RU/UZ/EN i18n (словарь + text-node walker)
- Модальная форма заявки с анимированным popup после отправки
- Модальное окно новости с полным текстом (5 готовых статей)
- Фильтры партнёров и новостей по категориям
- Адаптивная вёрстка (mobile-first)
- Material Symbols иконки
- Без зависимостей (чистый HTML/CSS/JS)

## Запуск локально

Открыть `index.html` в браузере или запустить любой static-server:

```bash
python -m http.server 8000
# или
npx serve
```

## Деплой

Совместим с Vercel, Netlify, GitHub Pages, Cloudflare Pages — без сборки.
